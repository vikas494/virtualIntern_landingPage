A sleek, high-conversion landing page built with a modern, dark-themed UI. This project features a responsive 50/50 split layout, fluid imagery, and embedded social icons—all accomplished with **strictly zero JavaScript**.

## ✨ Features

* **100% Responsive Design:** Built with modern CSS (Flexbox and Media Queries) to ensure a flawless layout across mobile phones, tablets, and massive desktop monitors.
* **Zero JavaScript:** Optimized for ultimate speed and simplicity using only pure HTML and CSS.
* **Fluid Image Scaling:** The hero section features a dynamic image that naturally scales and collapses perfectly based on the browser window size.
* **Embedded SVG Icons:** Crisp, scalable vector graphics for Instagram, LinkedIn, and X (Twitter) in the footer—no bulky external icon libraries required.
* **Modern UI/UX:** Features a deep dark background (`#111216`) accented with eye-catching neon green (`#ccff00`) calls-to-action to maximize user engagement.

## 🚀 Quick Start

Since this project uses pure HTML and CSS, there is no complex build process or installation required.

1.  Clone this repository or download the files.
2.  Ensure `index.html`, `index.css`, and your image (e.g., `teacher.jpeg`) are in the same directory.
3.  Double-click `index.html` to open it in your default web browser.
